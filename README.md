# CineGuesser

A modern, aesthetic movie guessing game built with Next.js and TypeScript.

## Getting Started

1.  **Install Dependencies:**
    Ensure you have Node.js installed. Then run:
    ```bash
    npm install
    ```

2.  **Run the Development Server:**
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to play.

## Features

-   **Genre Selection:** Choose from Action, Comedy, Sci-Fi, or Random.
-   **Game Loop:**
    -   You have 60 seconds to identify the movie from a frame.
    -   Use the **+20s** and **-20s** buttons to jump forward/backward in time (max 3 jumps).
    -   Type the movie title or character name to guess.
    -   Instant feedback on correct answers.
-   **Aesthetic UI:** Modern glassmorphism design with smooth animations.

## Tech Stack

-   Next.js 14 (App Router)
-   TypeScript
-   CSS Modules (No external UI libraries)
